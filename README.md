# Webpack React Seed  

This is a Webpack application seed configured with the following:  
* Webpack: v4.30.0
* React: v16.8.6
* @babel: v7.x.x

Configured with:  
* Linting via eslint
* Handlebars html templating
  * Custom Block Helpers
    * `#ifProduction`
    * `#ifDevelopment`
  * Custom Partials
    * Meyer CSS Reset
    * MetaData tags
      * Twitter Card
      * OpenGraph
      * Schema.org
    * Google Analytics

Additional Website Configuration in: 
* `site-config.json`
